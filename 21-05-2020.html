CSS Multiple background Property

this div contains two background-image(paper and flower).

  • CSS Property Used:
    • background-image: url('assets/images/flower.gif'), url('assets/images/paper.gif');
    • background-position: bottom right, top left;
    • background-repeat: no-repeat, repeat;
    • padding: 5px 10px;
    • height: 330px;
      • Or By Shorthand Property:
    • background:
      url('assets/images/flower.gif') bottom right no-repeat,
      url(assets/images/paper.gif) top left repeat;

NOTE: First background-image is closest the viewer.

background-size Property

this image have background-size: 200px width and 150px height.

  • CSS Property Used:
    • background-image: url('assets/images/flower.gif');
    • background-size: 200px 150px;
    • background-position: bottom right;
    • background-repeat: no-repeat;
      • Or By Shorthand Property:
    • background:
      url('assets/images/flower.gif') bottom right /
      200px 150px no-repeat;
