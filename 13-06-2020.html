Date()

JavaScript Date() Object

JavaScript Date Output :

Note: By default, JavaScript will use the browser's time zone and display a date as a full text string:

JavaScript Code Used : 

        var date00 = new Date();
        document.getElementById("demo-00").innerHTML = date00;

Creating Date Objects

There are 4 ways to create Date object.

  • new Date()
  • new Date(year, month, date, hour, minute, second, and millisecond)
  • new Date(millisecond)
  • new Date(date String)

new Date(year, month, ...) :

Note : new Date(year, month, day, hour, minute, second, and millisecond), these 7 property is Specified in same order.

JavaScript Code Used : 

        var date01 = new Date(2020, 07, 25, 8, 30, 30, 30);
        document.getElementById("demo-01").innerHTML = date01;

Note : JavaScript count Month from 0 to 11. 0 is January and 11 is December.
6 Number of property only specified(year, month, date, hour, minute, second).
5 Number of property only specified(year, month, date, hour, minute).
4 Number of property only specified(year, month, date, hour).
3 Number of property only specified(year, month, date).
2 Number of property only specified(year, month).
but 1 number of Property doesn't specified year, it specified millisecond.

For Example: Here var d = new Date(2020); this means 2020 is millisecond not year.

JavaScript Code Used : 

        var date02 = new Date(2020);
        document.getElementById("demo-02").innerHTML = date02;

Previous Century :

one and two digit year will taken as 19xx.

For Example: var d = new Date(20, 07, 25, 8, 30, 30, 30); and var d1 = new Date(0, 07, 25, 8, 30, 30, 30); Here 20 and 0 is treated 1920 and 1900 respectively.

JavaScript Code Used : 

        var date03 = new Date(20, 07, 25, 8, 30, 30, 30);
        document.getElementById("demo-03").innerHTML = date03;
		
        var date04 = new Date(0, 07, 25, 8, 30, 30, 30);
        document.getElementById("demo-04").innerHTML = date04;

new Date(dateString) :

JavaScript Code Used : 

        var date05 = new Date("2020, february, 29, 12:30:59");
        document.getElementById("demo-05").innerHTML = date05;

or

        var date05 = new Date("2020 february 29 12:30:59");
        document.getElementById("demo-05").innerHTML = date05;

or

        var date05 = new Date("february 29 12:30:59 2020");
        document.getElementById("demo-05").innerHTML = date05;

Note: There is no rule of order in writing date string, you can put month first or year first doesn't matter. we can also omit comma(,) but put space between each properties.

new Date(milliseconds) :

JavaScript stores dates as number of milliseconds since January 01, 1970, 00:00:00 UTC (Universal Time Coordinated).

Now the time is: milliseconds past January 01, 1970.

JavaScript Code Used : 

        function printMillisecond() {
          var date06 = new Date("2020, february, 29, 12:30:59");
          var d = date06.getTime();
          document.getElementById("demo-06").innerHTML = d;
        }

Example : var d = new Date(0); means : Thu Jan 01 1970 05:30:00 GMT+0530 (India Standard Time) .

JavaScript Code Used : 

        var date07 = new Date();
        document.getElementById("demo-07").innerHTML = date07;

Notes : Negative value of millisecond accept in javascript.
Example : var d = new Date(-100000000000) is approx.. Mon Oct 31 1966 19:43:20;

Displaying Dates :

By default javascript output dates in full string format. this means that when we display date object in html. its automatically converted to a string, with the toString() method.

both is Same

JavaScript Code Used : 

       function printDateString() { 
         var date08 = new Date();
         document.getElementById("demo-08").innerHTML = date08;
        
         var date09 = new Date();
         document.getElementById("demo-09").innerHTML = date09.toString();
       }
       setInterval(printDateString, 1000);

toUTCString()

JavaScript Code Used : 

         var date10 = new Date();
         document.getElementById("demo-10").innerHTML = date10.toUTCString();

toDateString() : This method converts date to more readable format.

JavaScript Code Used : 

         var date11 = new Date();
         document.getElementById("demo-11").innerHTML = date11.toDateString();
