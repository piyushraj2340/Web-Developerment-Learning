JavaScript Date() Object

JavaScript Date Output :

Note: By default, JavaScript will use the browser's time zone and display a date as a full text string:

JavaScript Code Used : 

        var date00 = new Date();
		
        document.getElementById("demo-00").innerHTML = date00;

Creating Date Objects

There are 4 ways to create Date object.

  • new Date()
  • new Date(year, month, date, hour, minute, second, and millisecond)
  • new Date(millisecond)
  • new Date(date String)

new Date(year, month, ...) :

Note : new Date(year, month, day, hour, minute, second, and millisecond), these 7 property is Specified in same order.

JavaScript Code Used : 

        var date01 = new Date(2020, 07, 25, 8, 30, 30, 30);
        
        document.getElementById("demo-01").innerHTML = date01;

Note : JavaScript count Month from 0 to 11. 0 is January and 11 is December.
6 Number of property only specified(year, month, date, hour, minute, second).
5 Number of property only specified(year, month, date, hour, minute).
4 Number of property only specified(year, month, date, hour).
3 Number of property only specified(year, month, date).
2 Number of property only specified(year, month).
but 1 number of Property doesn't specified year, it specified millisecond.

For Example: Here var d = new Date(2020); this means 2020 is millisecond not year.

JavaScript Code Used : 

        var date02 = new Date(2020);
		
        document.getElementById("demo-02").innerHTML = date02;

For Example: var d = new Date(20, 07, 25, 8, 30, 30, 30); and var d1 = new Date(0, 07, 25, 8, 30, 30, 30); Here 20 and 0 is treated 1920 and 1900 respectively.

new Date(milliseconds) :

JavaScript stores dates as number of milliseconds since January 01, 1970, 00:00:00 UTC (Universal Time Coordinated).

Now the time is: milliseconds past January 01, 1970.

JavaScript Date Formats

JavaScript Date Input :

There are generally 3 type of JavaScript date input formats:
  • ISO Date -- Example: "2020-06-14"(The International Standard)
  • Short Date -- Example: "06/14/2020
  • Long Date -- Example: "Jun 14 2020" or "14 Jun 2020"

Notes: The ISO format a strict standard in JavaScript. The other format are not so well defined and might be browser specific.

JavaScript ISO Dates & Time :

Notes:

ISO 8601 is the international standard for the representation of dates and times.

The ISO 8601 syntax (YYYY-MM-DD) is also the preferred JavaScript date format:

& & & & &

JavaScript Code Used : 

         var date12 = new Date("2020-06-14"); // year, month and date.
		   
         document.getElementById("demo-12").innerHTML = date12;
		 
		 // & 
		 
		 var date13 = new Date("2020-06"); // year and month.
		   
         document.getElementById("demo-13").innerHTML = date13;
		 
		 // & 
		 
		 var date14 = new Date("2020"); // only year.
		   
         document.getElementById("demo-14").innerHTML = date14;
		 
		 // & 
		 
		 var date15 = new Date("2020-06-14T12:00:00Z"); // date and time. related to UTC.
		   
         document.getElementById("demo-15").innerHTML = date15;
		 
		 // & 
		 
		 var date16 = new Date("2020-06-14T11:15:00+05:30"); // date and time. related to UTC +05:30.
		   
         document.getElementById("demo-16").innerHTML = date16;
		 
		 // & 
		 
		 var date17 = new Date("2020-06-14T11:15:00-05:30"); // date and time. related to UTC -05:30.
		   
         document.getElementById("demo-17").innerHTML = date17;

Note :
Here capital T Separate Date and time.

UTC time is defined with a Capital Letter Z.

we can also modify the time related to different time zone, Remove the Letter Z and add +HH:MM or -HH:MM instead:

Date Get Methods

Method Description
getFullYear() Get the year as a four digit number (yyyy).
getMonth() Get the month as a number (0-11).
getDate() Get the day as a number (1-31).
getHours() Get the hour (0-23).
getMinutes() Get the minute (0-59).
getSeconds() Get the second (0-59).
getMilliseconds() Get the millisecond (0-999).
getTime() Get the time (milliseconds since January 1, 1970).
getDay() Get the weekday as a number (0-6).
Date.now() Get the time. ECMAScript 5.
