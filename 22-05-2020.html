CSS Gradient(linear and radial)

Syntex:
background-image: linear-gradient(direction, color-stop1, color-stop2, ...);
background-image: radial-gradient(shape size at position, start-color, ..., last-color);

linear-gradient

linear-gradient-- direction Top To Bottom (This is default)

  • CSS Property Used:
    • background-image: linear-gradient(red,yellow);
    • height: 300px;
    • padding: 10px 20px;

repeating-linear-gradient

  • CSS Property Used:
    • background-image: repeating-linear-gradient(red, yellow 10%, green 20%);
    • height: 300px;
    • padding: 10px 20px;

radial-gradient

Evenly spaced color stop(this is default)

  • CSS Property Used:
    • background-image: radial-gradient(red,yellow,green);
    • height: 300px;
    • padding: 10px 20px;
    • width: 100%;
    • margin: 0;

Use of Different Size Keywords

ellipse:
circle :

    • background-image: radial-gradient(farthest-side at 55% 65%, red, yellow, green);
